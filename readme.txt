This plugin allows to subscribe users to different Mailchimp lists based on the product that they buy.

== Description ==

This Plugin will add to every woocommerce product a new field called Mailchimp List. 
This will allow you to subscribe users to different lists based on the products the're buying.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tools->Chimp Mail List By Woo Product screen to configure the Mailchimp Api Key

You can contact us at pluginwp@smartservices.com.ar or visit <a href="https://www.smartservices.com.ar">our site</a>.


== Frequently Asked Questions ==

= How can I set the List for each product? =

In the products, there's a new field called Mailchimp List. Set the Mailchimp List ID there and that's all. The ID of the list, no the name.
